**run a postgres database using docker compose:**

db host: `localhost`

database name: `postgres`

port: `5432`

username: `postgres`

password: `password`

```shell
docker compose up -d
```

**Run dart_forg server:**

```shell
dart pub get

dart_frog dev
```
